# test-termosphere

Тестовое задание: первый аналитический контур над выгрузкой из CRM (Bitrix24-подобной) и связанных производственных таблиц.

**Пайплайн:** CSV/Excel → PostgreSQL → нормализация (raw → core) → SQL-отчёты → API / дашборд.

## Содержимое

- `Тестовое задание .pdf` — исходное ТЗ.
- `BRIEF.md` — разбор задачи: бизнес-логика, модель данных (ERD/BPM/FSM), каталог проблем качества данных, принятые решения, план внедрения.
- `data/` — исходные данные в CSV (добавляются отдельно).

## Стек (план реализации)

FastAPI · SQLAlchemy · Alembic · PostgreSQL (Docker) · React (Vite) · Mermaid (ERD/BPM/FSM).

## Статус

Подготовка. Реализация — по плану внедрения из `BRIEF.md`.
